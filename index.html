How It Works

Install both, hear nearby

Both players need TileWhisper installed. When you're on the same OSRS world within 15 tiles (configurable), you can hear each other speak. Volume fades smoothly as you walk apart.

01

Install Plugin

Get TileWhisper free from RuneLite Plugin Hub. Restart RuneLite — it just works.

02

Configure Mic

Open TileWhisper settings, choose Push-to-Talk or Voice Activity, set your key or sensitivity.

03

Join World

Any player on the same world with TileWhisper is in your voice channel. No invites, no friend requests.

Features

Built for OSRS proximity chat

🎤

Push-to-Talk or Auto

Hold V (configurable) to broadcast, or use Voice Activity Detection to auto-transmit when speaking.

📍

Proximity Audio

Volume decreases linearly with Chebyshev distance — same math OSRS uses for tile movement. Feels natural.

🔇

Per-Player Controls

Mute or adjust volume (0–200%) for each player individually. Speaking indicators show who's talking.

💻

Cross-Platform

Pure Java Opus codec works on Linux, macOS (including Apple Silicon), and Windows equally well.

Privacy & Transparency

We see exactly what we need to relay

What the relay server sees

👤

Your OSRS username

📍

Approximate tile coordinates (world, x, y, plane)

🎤

Encoded audio (Opus at 32 kbps)

What we don't do:

❌

Nothing is stored or logged

🌐

Audio is never broadcast globally — only to players on your world within range

🔐

No accounts, no signups, no tracking

Server source code is public. Both plugin and server are free to use and self-host if you prefer.
